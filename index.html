FAQ

Straight answers.

How much will I actually earn?

Coffee money to start — not rent. Heavy AI users see a few dollars a month early on. As advertiser demand grows, so does your cut. We'll publish network averages publicly as the network scales.

What does Lull actually see?

Nothing you type. Lull detects only the loading state — never prompts, code, or responses. The source is on GitHub. Don't take our word for it — read it.

Which tools are supported?

VS Code and Cursor via the IDE plugin — v0.1 scope. Browser support is on the public roadmap and ships next.

How do I cash out?

USDC to any wallet, or Stripe to your bank. Cash out anytime above $10. No points, no lock-ins.

Why not the VS Code Marketplace?

Marketplace policy prohibits ad-based extensions. We distribute via GitHub Releases — same install flow, one extra click. We ship faster and can never get de-listed.
